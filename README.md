Projeto de Playbooks de Teste - Ansible

Este repositório tem como objetivo centralizar todos os playbooks de teste utilizados para estudo, validação e desenvolvimento de automações com Ansible.
Aqui serão armazenadas automações voltadas para:
Provisionamento de servidores
Configuração de sistemas operacionais
Integração com serviços de infraestrutura
Instalação de aplicações
Hardening básico
Testes de roles e módulos
Simulações de cenários corporativos

O foco é servir como:

✔ Laboratório de aprendizado
✔ Base para automações futuras
✔ Ambiente de validação antes de produção
✔ Repositório de boas práticas
